Emoji cheat sheet for Campfire and GitHub

Emoji emoticons is supported on 37signals Campfire and on GitHub. However the services doesn't support the same set of emojis and some of the codes are not super easy to remember. So here is a little cheat sheet.
✈ If you got flash enabled click the emoji code and it will be copied to your clipboard.
